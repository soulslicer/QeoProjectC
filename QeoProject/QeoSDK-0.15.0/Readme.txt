QEO SDK DELIVERY RELEASE 0.15.0
---------------------------------------

Qeo is a platform indepedent protocol for inter-device communication. This SDK enables developers to develop 
Qeo-enabled applications. This version of the SDK supports development on the Android platform as well as development in C and JavaScript.


RELEASE IDENTIFICATION
-----------------------

* ID: Qeo SDK
* Version number: v 0.15.0
* Description: Technicolor Qeo SDK Version 0.15.0


FILE  STRUCTURE
---------------

This release contains the following files and folders:

FILES IN ROOT:

* Readme.txt ------------------------------------------------ This file
* Installation Guide.PDF ------------------------------------ How to Install the SDK
* Copyright and Confidentiality Information Notice.pdf ------ Copyright and Confidentiality Information Notice
* Licensing Information.pdf --------------------------------- Licensing Information

FOLDERS:

/android ---------------------------------------------------- Contains all SDK files for development in Android
/c ---------------------------------------------------------- Contains all SDK files for development in C
/js --------------------------------------------------------- Contains all SDK files for development in JavaScript
/qdms ------------------------------------------------------- Contains all files related to the Qeo Data Model
/tools ------------------------------------------------------ Contains the Qeo code generator for generating code from a QDM file

For more information on these folders and their subfolders, refer to the Installation Guide. This document is located in the root of the SDK package (same as this document)
as well as on the documentation website http://wiki.developer-partnersalliance.com/display/QeoCusDoc/Customer+Documentation+Space+Home
