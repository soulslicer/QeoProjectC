Auflistung der Klassen
Hier folgt die Aufzählung aller Klassen, Strukturen, Varianten und Schnittstellen mit einer Kurzbeschreibung:
gnublin_adcKlasse für den zugriff auf die GPAs
gnublin_csvKlasse zum Erstellen von csv Dateien
gnublin_gpioKlasse für den zugriff auf die GPIO Pins
gnublin_i2cKlasse für den zugriff auf den GNUBLIN I2C Bus
gnublin_module_adcKlasse für den Zugriff auf das GNUBLIN MODULE-ADC
gnublin_module_dacKlasse für den zugriff auf den MCP4728 digital zu analog Wandler
gnublin_module_dogmKlasse für den Zugriff auf das GNUBLIN MODULE-DISPLAY 2x16
gnublin_module_lcdKlasse für den zugriff auf das GNUBLIN Module-LCD 4x20
gnublin_module_lm75Klasse für den zugriff auf den LM75 IC via I2C Bus
gnublin_module_pca9555Klasse für den zugriff auf den PCA9555 IC via I2C Bus
gnublin_module_relayKlasse für den zugriff auf das GNUBLIN module-relay
gnublin_module_stepKlasse für den zugriff auf das GNUBLIN module-step
gnublin_pwmKlasse zum Erstellen von pwm Signalen
gnublin_serialKlasse für den zugriff auf die serielle Schnittstelle des GNUBLIN
gnublin_smtpKlasse um Mails zu versenden
gnublin_spiKlasse für den Zugriff auf den SPI-Bus
