int gnublin_serial::send ( unsigned char *  TxBuf,
int  length 
)

sendet Bytes an den serielle Schnittstelle.

Diese Funktion sendet "length" Anzahl an Bytes aus dem "TxBuf" an die serielle Schnittstelle. Bei Erfolg wird 1 zurück gegeben, bei Misserfolg -1.
Beispiele:
Sende 2 Bytes von "buf" an den i2c Bus: send(buf, 2);

Parameter:
RxBufSende Puffer. Die zu sendenden Bytes sind hier gespeichert.
lengthAnzahl der zu sendenden Bytes.
Rückgabe:
Erfolg: 1, Misserfolg: -1
